# Pointers Exercises

## Exercise 1: Array Manipulation
Write functions using pointers to manipulate arrays.

**Requirements:**
- Find maximum element using pointers
- Reverse array using pointers
- Find sum of all elements

## Exercise 2: String Operations
Implement string functions using pointers.

**Requirements:**
- String length without strlen()
- String copy without strcpy()
- String concatenation without strcat()
- String comparison without strcmp()

## Exercise 3: Dynamic Array
Create a dynamic array implementation.

**Requirements:**
- Initialize with given size
- Resize when needed
- Insert and delete elements
- Free memory properly

## Exercise 4: Pointer Arithmetic
Practice pointer arithmetic operations.

**Requirements:**
- Navigate through array using pointer arithmetic
- Calculate difference between pointers
- Access 2D array using pointer arithmetic

## Exercise 5: Function Pointers
Use function pointers for callback mechanisms.

**Requirements:**
- Create array of function pointers
- Implement calculator using function pointers
- Sort array with custom comparator
