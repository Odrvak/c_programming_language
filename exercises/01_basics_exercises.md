# Basics Exercises

## Exercise 1: Temperature Converter
Write a program that converts temperature from Celsius to Fahrenheit and vice versa.

**Requirements:**
- Ask user for temperature value
- Ask for conversion type (C to F or F to C)
- Display result with 2 decimal places

## Exercise 2: Simple Calculator
Create a calculator that performs basic arithmetic operations.

**Requirements:**
- Accept two numbers and an operator (+, -, *, /)
- Display the result
- Handle division by zero

## Exercise 3: Prime Number Checker
Write a program that checks if a number is prime.

**Requirements:**
- Accept a number from user
- Check if it's prime
- Display appropriate message

## Exercise 4: Fibonacci Series
Generate Fibonacci series up to n terms.

**Requirements:**
- Accept number of terms from user
- Display the series
- Use both iterative and recursive approaches

## Exercise 5: Pattern Printing
Print various patterns using loops.

**Requirements:**
- Print a right triangle of stars
- Print a pyramid of stars
- Print a hollow square
